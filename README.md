# SteakRoll
### World of Warcraft: Wrath of the Lich King 3.3.5a

Automatically roll on certain conditions to cut down on repetative rolling.

* Automatically roll disenchant/greed on greens.
* Automatically roll disenchant/greed on anything you can not roll need on.

I wouldn't recommend using this on a toon while leveling when a green might actually be an upgrade you want to need on.
